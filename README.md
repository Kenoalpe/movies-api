# Getting started!
1. **Clone**
    
`git clone https://github.com/Kenoalpe/movies-api.git`

`cd movie-api`

2. **Run**
   
`dotnet run --project Kenoalbers.Movies.Api/ --launch-profile http`

or

`docker compose up`

3. **OpenApi documentation**
   
Open `http://localhost:8080/swagger` in a webbrowser.
